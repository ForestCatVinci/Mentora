СТРАНИЦЫ
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

LANDING
Красивая страница с описанием платформы. Кнопки "Войти" и "Зарегистрироваться".
Современный дизайн, фиолетовый акцент #7C3AED.

ONBOARDING
Wizard из трёх шагов. Без завершённого онбординга лента недоступна.
  Шаг 1: выбор класса — большие кликабельные карточки с цифрами 8, 9, 10, 11
  Шаг 2: выбор интересов мультиселект:
    STEM, Business, CS, Math, Language, Research,
    Olympiads, Entrepreneurship, Science, Social Impact
  Шаг 3: выбор целей:
    Поступление за рубеж, Олимпиады, Стартап, Исследования
После завершения пишем в таблицу users поля grade, interests, goals.
Редирект на /feed.

NAVBAR
Иконки + подписи. Вкладки: Лента, Возможности, Курсы, Roadmap, Менторы, Личный кабинет.
Активная вкладка выделена фиолетовым.
Для staff/admin дополнительно показываем кнопку "Создать пост".

FEED
GET http://localhost:8080/feed/{user_id}
Показываем PostCard для каждого поста.
Если пустая — empty state с призывом обновить интересы в профиле.

POSTCARD (компонент)
  - Изображение
  - Title
  - summary_ru (краткое описание от AI)
  - Первые 3 тега как бейджи
  - Category badge
  - Deadline с цветовой индикацией:
      <= 7 дней  → красный бейдж "X дней осталось"
      <= 42 дней → жёлтый бейдж
      > 42 дней  → серый, просто дата
  - Кнопка сохранить (иконка закладки, toggle)
      POST /posts/{id}/save или DELETE /posts/{id}/save

OPPORTUNITIES
Все опубликованные посты. GET http://localhost:8080/posts
Фильтры по category и grade. Поиск по title.

COURSES
GET http://localhost:8080/courses
CourseCard: image, title, description, difficulty badge,
прогресс-бар из user_progress, кнопка "Начать" или "Продолжить".

ROADMAP
Статичная страница. Визуальный timeline: 8 → 9 → 10 → 11 класс.
Каждый год содержит чекбоксы с активностями
(олимпиады, хакатоны, летние школы, SAT prep и т.д.).
Текущий класс ученика подсвечен акцентом.
Чекбоксы сохраняются в localStorage.

MENTORS
GET http://localhost:8080/mentors
MentorCard: аватар, имя, университет, специальность,
кнопка "Записаться" (открывает mailto).

DASHBOARD
Личный кабинет студента.
Показывает: имя, класс, интересы.
Список сохранённых постов.
Прогресс по курсам.
Кнопка редактировать профиль (интересы и цели).

CREATE POST (только staff/admin)
Форма: title, description (textarea), image (file upload с превью),
deadline (DatePicker — всплывающий календарь).
POST multipart/form-data на http://localhost:8080/posts
Показывай лоадер пока OpenAI теггирует (~2-3 сек).
После успеха показать созданный пост с тегами.

ДИЗАЙН — КРИТИЧНО
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Интерфейс должен выглядеть как реальный продукт, не учебный проект.
Целевая аудитория — школьники 8-11 классов.

  Основной цвет: фиолетовый #7C3AED
  Шрифт: Inter (подключи через Google Fonts)
  Карточки с мягкими тенями и hover эффектами (scale + shadow)
  Плавные transitions 150-200ms везде
  Скелетон-лоадеры вместо спиннеров
  Адаптивная вёрстка: мобильный + десктоп
  Empty states с иконкой и текстом
  Онбординг как красивый wizard с прогресс-баром сверху

Не делать:
  Дефолтный серый Tailwind без кастомизации
  Таблицы там где должны быть карточки
  Плотную вёрстку без отступов
